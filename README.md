
# Sample Drag and Drop

Sample Drag and Drop using Typescript


## Installation

Clone my project with git
```bash
  git clone https://github.com/kareem1415/sample-drag-and-drop.git
```
install it by npm

```bash
npm install
or
yarn add
```
to run my project

```bash
npm start
```
to get build with webpack

```bash
npm run build
```
